## 1. Purpose

`ASH-TCU-L16C` proves that one logical `16x16` software matrix tile can be executed through native WGPU dispatch and read back as raw `1024` bytes, equal to `256 f32` values.

L16C depends on the L16A coordinate contract and the L16B CPU mirror parity seal. It compares native GPU readback against CPU reference GEMM under tolerance.

This patch opens only a single logical16 native dispatch. It does not open KPanel4, K6B 4x4 macro grid dispatch, production dispatch, runtime splice, or performance selection.
