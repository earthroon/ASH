## 1. Purpose

`ASH-TCU-K6B-R0` adapts the K6A square `4x4` macro descriptor plan into sixteen K6B logical16 KPanel4 macro control records.

It is a control bridge patch, not a compute patch. It validates descriptor shape, macro grid shape, macro offsets, base element formulas, output layout, and double-offset guards before K6B native `4x4` dispatch is opened.

## 7. Double Offset Guard

`row_offset` and `col_offset` are metadata. Base elements already include the offset.

Allowed A index:

```text
a_index = a_base_element + local_row * 32 + kk
```

Forbidden A index:

```text
a_index = a_base_element + row_offset * 32 + local_row * 32 + kk
```

Allowed B index:

```text
b_index = kk * 64 + b_base_element + local_col
```

Forbidden B index:

```text
b_index = kk * 64 + b_base_element + col_offset + local_col
```
