# QW-55A-0-R2A
## Logical VTC16 Native WGPU Dispatch Receipt / Adapter Fingerprint Readback Parity Seal

## Scope

QW-55A-0-R2A is the native WGPU dispatch proof layer for the QW-55A-0-R2 Logical16FromTile8SoftGroup shader.

R2 sealed the single-dispatch logical VTC16 kernel in static CPU mirror mode. R2A must request a real WGPU adapter, create a device and queue, dispatch the exact R2 shader, submit the command buffer, read back the C atlas, and compare GPU output against the CPU reference.

This patch is not a performance benchmark and must not open runtime inference authority.

## Parent seal

```text
parent_patch_id = QW-55A-0-R2
kernel_kind = Logical16FromTile8SoftGroup
shader_path = crates/burn_webgpu_backend/src/shaders/tensorcube_atlas_microtile_16x16_from8x8_softgroup.wgsl
workgroup_size = [64,1,1]
physical_tile_shape = [8,8]
logical_tile_shape = [16,16]
```

R2A must dispatch the exact R2 shader source. If the shader hash does not match the parent hash, PASS is forbidden.

## Required native sequence

```text
request_adapter
request_device
create A storage buffer
create B storage buffer
create C storage buffer
create Macro16Desc storage buffer
create readback staging buffer
create shader module from R2 shader
create bind group layout
create compute pipeline
create bind group
encode compute pass
dispatch_workgroups(macro_desc_count, 1, 1)
copy C buffer to staging buffer
queue.submit
device.poll or equivalent wait path
map_async staging readback
compare GPU C tiles against CPU reference
emit receipt
```


Strict mode:

