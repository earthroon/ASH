# QW-55A-0-R3
## Logical VTC16 General-K Panel Accumulation / TileId Table Offset Seal

## Scope

QW-55A-0-R3 generalizes the QW-55A-0-R2A native-dispatched Logical16FromTile8SoftGroup kernel from fixed `k_panels = 2` into a General-K panel accumulation candidate.

R3 adds a new shader and must not mutate the R2/R2A parent shader:

```text
crates/burn_webgpu_backend/src/shaders/tensorcube_atlas_microtile_16x16_from8x8_softgroup_kpanel.wgsl
```

The output remains four physical 8x8 Vec4F32 atlas tiles. No persistent physical 16x16 tile is created.

## Descriptor / control buffer contract

R3 uses a single storage control buffer instead of a fifth storage binding so that adapters reporting `max_storage_buffers_per_shader_stage = 4` remain valid.

```text
binding(3) control_words:
[0] a_tile_table_offset
[1] b_tile_table_offset
[2..5] c_tiles [C00,C01,C10,C11]
[6] k_panel_count
[7] reserved
[8..] tile_id_table
```

A tile table layout:

```text
a_index = a_tile_table_offset + row_block * k_panel_count + kp
```

B tile table layout:

```text
b_index = b_tile_table_offset + kp * 2 + col_block
```
