# QW-52A Spec Snapshot

QW-52A is a Rust-owned side-channel trace patch. It binds existing HangulFeatureRow, QWave cell, pulse, and transition modules to top-k decode candidates. It forbids new math rewrites, logit mutation, sampler mutation, token ban, hidden-state fusion, adapter projection, WebGPU backend switching, and Python-owned analysis.
